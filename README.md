# password_manager
NNTM - NoNeedToMemorise
NNTM is a password manager application to store login credentials

# Author
Anika Varsha Shekar - @AnikaShekar

# Features
- Password masking & show/hide toggle
- Copy to clipboard
- Delete functionality
- Duplicate prevention
- Responsive
